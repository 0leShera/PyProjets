Управлять Flask Server можно через терминал.
Все команды даются через терминал.

Для начала необходимо установить библиотеку virtualenv.
pip install virtualenv

Создать окружение на локальной машине, указав его имя.
Например, как у меня, venv.
virtualenv venv

Как только окружение создано, активировать его:
source venv/bin/activate
Чтобы деактивировать окружение, ввести команду deactivate

Экспортировать в окружение файл сервера:
export FLASK_APP = main_flask.py 

Запустить сервер:
flask run --host="0.0.0.0" --port="5077"
# номер хоста для примера
# номер порта для примера

После внесения изменений в файл main_flask.py и их сохранения
Нажать ctrl + c. Затем заново запустить командой:
flask run --host="0.0.0.0" --port="5077"

export FLASK_ENV=development
